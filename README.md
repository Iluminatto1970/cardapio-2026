# Cardápio 2026

SaaS de cardápio digital com checkout, autenticação e integração WhatsApp.

## Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS + shadcn/ui
- Prisma + SQLite
- Vitest

## Setup

```bash
npm install
npm run db:push
npm run dev
```

## Variáveis de Ambiente

Copie `.env.example` para `.env` e configure.

## Deploy

Vercel recomendado para Next.js.
